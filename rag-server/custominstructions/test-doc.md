# Test Documentation

This is a test document for the RAG server.

## Features

1. **Search Functionality**: The server can search through documentation
2. **Question Answering**: Ask questions about the codebase
3. **Code Indexing**: Index Python, TypeScript, and JavaScript files

## Configuration

The server uses:
- Qdrant for vector storage
- Sentence transformers for embeddings
- Hybrid search (BM25 + Vector)

## Testing

This document is used to test:
- Indexing capabilities
- Search functionality
- Question answering
- Document retrieval
